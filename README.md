# dozy-tech-demo

Tech demo of the [dozy framework](https://github.com/dozy-framework/dozy).

## Setup

1. Ensure that you meet all the prerequisites required by the dozy framework.
2. Clone this repo locally, including its submodules:

```
git clone --recurse-submodules git@github.com:dozy-framework/dozy-tech-demo.git
```

## Running

To run this demo, do:

```sh
zig build run
```
